Label dictionary and analysis objects

The PDF analysis uses a small set of shared objects defined in R/Analysis setup.R. These help standardize variable labels across tables and figures and store supporting information used in the results pages.

Variable label dictionary

The label_list object maps internal variable names used in the analysis code to reader-friendly labels used in tables and summaries. This allows the same underlying variables to be displayed consistently across outputs without manually renaming them in each table.
